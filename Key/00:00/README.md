本节使用最简单的按键方式--外部中断按键

在常见的STC系列单片机中，外部中断相关引脚通常被放在P3^2，P3^3中。
这里使用P3^2对应的外部中断EX0，它的中断向量号为0，同时拥有最高的优先级

首先是配置外部中断并使能：

IT0 = 1;  //配置模式寄存器为下降沿触发，IT0 = 1为上升沿触发
EX0 = 1;  //使能外部中断0，External 0
EA = 1;   //总中断使能，Enable All

配置完成后，还要编写中断函数:

void EX0_Isr()  interrupt 0
{}
